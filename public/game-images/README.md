# Game image sets

The `/game` page loads images from three places, with a fall-through:

| Slot | Where it lives | Distributed publicly? | Use case |
|------|----------------|------------------------|----------|
| **`real/`** | local-only (gitignored) | ❌ | Populated from FFHQ / 140k real. Local install only. |
| **`fake/`** | local-only (gitignored) | ❌ | Populated from 140k StyleGAN fake. Local install only. |
| **`fallback-fakes/`** | committed | ✅ | 5 sample StyleGAN faces — used when local `fake/` is empty (e.g. on Vercel). No real person, so no privacy/redistribution concern. |
| **`manifest.json`** | local-only (gitignored) | ❌ | Generated by the populate script; lists what's in `real/` and `fake/`. |

The real-face fallback for the public demo is a curated set of Unsplash
URLs hard-coded in `src/app/game/page.tsx` — no images are committed for
that path.

## Why this split

The 140k dataset contains real faces from FFHQ (Flickr). FFHQ is
CC BY-NC-SA-licensed and built from photos whose subjects did not
explicitly consent to being used in deepfake-detection challenges.
For a public-facing site that's an ethical grey area; keeping those
images out of the repository sidesteps it entirely.

The StyleGAN-generated fakes are synthetic — no person to ask. They
ship in `fallback-fakes/` so the public demo still works.

## Populating the local sets

```powershell
# From repo root
python engine\scripts\build_game_dataset.py `
  --source-real "C:\path\to\140k\test\real" `
  --source-fake "C:\path\to\140k\test\fake" `
  --count 20
```

The script copies `count` random images from each source folder into
`public/game-images/{real,fake}/`, then writes a `manifest.json` so the
frontend can discover them without an extra API call.

## Adding the fallback fakes (5 images, one-time)

1. Pick 5 representative StyleGAN faces from your 140k fake set.
2. Copy them into `public/game-images/fallback-fakes/`.
3. Name them `fake_01.jpg`, `fake_02.jpg`, …, `fake_05.jpg`.
4. `git add public/game-images/fallback-fakes/ && git commit`.

These ARE committed; ensure they truly are synthetic (no real face).
